CoT Math Solver\n================\n\nThis small project demonstrates a Chain-of-Thought (CoT) math solver using the OpenAI API. It instructs the model to return structured JSON containing step-by-step reasoning, then verifies arithmetic locally.\n\nFiles created:\n- `cot_math_solver.py` — main Python script that calls the OpenAI ChatCompletion API, parses JSON CoT, safely evaluates expressions, verifies results, and prints reasoning + final answer.\n- `cot_requirements.txt` — minimal requirements for this component.\n\nQuick start (PowerShell)\n------------------------\nSet API key and run the example:\n```powershell\n$env:OPENAI_API_KEY = \"sk-...\"\npython cot_math_solver.py\n```\n\nNotes:\n- The script expects `OPENAI_API_KEY` to be set.\n- It uses a strict system prompt to force JSON-only output. If JSON parsing fails, the raw model output is printed.\n- The script uses a safe AST evaluator and a reconciliation loop on mismatches.\n\nIf you'd like, I can:\n- Add a CLI wrapper to accept arbitrary problems from the command line.\n- Add unit tests for `safe_eval` and the parsing logic.\n- Switch to the newer Responses API if desired.\n
